אפליקציית טרמינל להצגת תמונת מצב לתחנת 
עצירה לרכבים במרוץ.
הלוח יציג את המרוץ והסיבוב עם פרטיו, הרכבים שממתינים להיכנס, הרכב הבא וקריאה לנהג הממתין.


מבנה הקבצים והפונקציות:

דאטה על המרוץ:
אובייקט
שם: שם
הקפה: הקפה
הקפות סהכ: הקפות סהכ

דאטה של רכבים:
רשימה ובתוכה אובייקטים, אובייקט:
מספר רכב: מספר רכב
שם נהג: שם נהג 
סטטוס: "done "/" waiting"

פונקציות:

APIservice.js:

loadFromAPI() - מביא את הדאטה מהשרת והופך לאובייקט ג'ייסון
saveDataToJson(data) - מקבל אובייקט ג'ייסון מפרק אותו לדאטה על המרוץ ועל הרכבים, וכותב אותם בפורמט תקין לקבצי הג'ייסון בהתאמה.


JsonService.js:

getCarsAndRaceData() - טוען את המידע מקבצי הג'ייסון ומחזיר אותו כאובייקט עם המידע על המירוץ ורשימת רכבים


carsMethods.js:
 כולם מקבלים את רשימת הרכבים ומחזירים מניפולציות עליה:

getNextCar(cars) - מחזיר את הרכב הבא בתור
getCarByNumber(cars, num) - מחזיר רכב לפי מספר
carsAlreadyPassed(cars) - מחזיר מספר רכבים שכבר סיימו
carFormat(car) - מקבל רכב ולא רשימה ומחזיר פורמט הדפסה שלו
printWaitingCars(cars) - מדפיס את כל הרכבים הממתינים בפורמט הרצוי


main.js:
main()
הצגת האפליקציה שמבצעת הצגה, טעינה מהשרת לג'ייסון, מהג'ייסון לאפליקציה והצגתם

הרצה באמצעות - node main.js
